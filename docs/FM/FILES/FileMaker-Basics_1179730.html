FileMaker Docs : FileMaker Basics

Created by Jose Bernal [Administrator], last modified on Apr 19, 2016

FileMaker Go - Login

We both installed FileMaker Go on our iOS devices and troubleshooted issues around correctly configuring the "host" and "file" options.  At the end of our meeting we enabled a FileMake Pro option that allows users to save their usernames and passwords in their OSX Keychains.  This features forgoes having to enter credentials each time a database file is opened.

FileMaker Go - Layouts

We explored the Counsel layout both on the desktop machine and our mobile phones and discussed some of the implications for designing layouts within the constraints of smaller devices that require narrower layouts.  These narrower layouts minimize panning, especially on phone devices, which can lead to confusion about where a user is on a layout. 

Layout mode

We discussed several features of layout mode, including duplicating objects (Apple D), moving objects, mouse shortcuts for selecting multiple objects, locking objects, and grouping objects.  We also adjusted the formatting of objects and applied bold, font and size adjustments to at least one object.  Finally, we reviewed the three major parts of each layout: header, body and footer and positioned objects within all three parts, including a "Last Updated" field.

FileMaker modes

We talked about the differences between the various modes in FileMaker Pro: browse, find, layout and preview.

FileMaker views

We also reviewed the differences between Form, List and Table view.

Layouts and naming

We discussed proper naming of layouts and tables to adequately reflect their purpose and to "self document" our work by inserting labels and comments directly onto layouts.

Shortcuts

We discussed various shortcuts for navigating within FileMaker Pro and tested taking screenshots of specific windows and menus using auxiliary keys like Shift Apple 4 and the Spacebar in Mac OS X.
