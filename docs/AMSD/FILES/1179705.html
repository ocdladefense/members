AMS DOCS : Donations: Getting Started

Created by Jose Bernal [Administrator], last modified on Apr 18, 2016

Central to accepting Donations are Campaign Pages.  A Donation Campaign Page provides an intuitive user interface for accepting gifts from donors by displaying Items specially configured as "Donations."  Once a Donation Campaign Page is published, the AMS eStore can link to the new page and accept both one-time and recurring donations.  But before a Campaign Page can be published some preliminary work has to be completed.  To prepare the Donation Campaign Page, there must be published Donation Items, together with their respective Item Classes; in addition, Items should be organized into Catalogs.

Creating a Donation Campaign Page - Preliminary Work

The full list of preliminary work includes:

  • Configuring and publishing an Organization's eStore
  • Adding relevant Donation Items
  • Organizing Donation Items into appropriate Item Classes
  • Creating Catalogs to house Donations Items
  • Publishing Donation Items into Catalogs
  • Creating and publishing a Donation Campaign Page
  • Linking the Donation Campaign Page so it is readily accessible
